    704. Binary Search

    Solution: two pointer

    Code:

    class Solution:
    class Solution:
    def search(self, nums: List[int], target: int) -> int:
        if len(nums) == 1:
            return 0 if nums[0] == target else -1
        
        l = 0
        r = len(nums) - 1
        
        while l <= r:
            mid = (l + r) // 2
            if nums[mid] > target:
                r = mid -1
            elif nums[mid] < target:
                l = mid + 1
            else:
                return mid
        return -1

    Time Complexity: O(logn) Space Complexity: O(1)
